# Saad Cheatsheet 🚀

**Saad Cheatsheet** is a collection of quick developer references, commands, tips, and useful snippets curated by **Muhammad Saad Amin** to speed up development and learning.

## 📚 About

Saad Cheatsheet is designed to be a **fast lookup guide** for developers. Instead of searching the internet repeatedly, you can quickly find commonly used commands, syntax, and tricks in one place.

This repository will continuously grow with new cheatsheets covering different technologies.

## 🎯 Goals

- Provide **quick references for developers**
- Save time while coding
- Keep frequently used commands in one place
- Build a **central knowledge base**

## 📂 Contents

This repository may include cheatsheets for:

- Git & GitHub
- Linux Commands
- JavaScript / TypeScript
- Node.js
- React / Next.js
- Docker
- DevOps
- VS Code
- and more...

## ⚡ Usage

Simply browse the folders or files to find the cheatsheet you need.

Example:

```bash
git clone https://github.com/USERNAME/saad-cheatsheet.git
cd saad-cheatsheet
```

Then open any cheatsheet file.
